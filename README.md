# AI 戰情室 Pro (AI Stock War Room)

基於 Node.js 的台股量化監控系統。

## 版本更新日誌

### [v8.4] - 2026-02-23
- **✨ 權證熱度列表**: 新增模仿權證高手介面的即時熱度監控表格。
- **💥 解除標的限制**: 現在支援 6 碼權證代號與 4 碼普通股之全量監控。
- **⚡ 即時量能模組**: 新增 `lib/realtime.js`，對接證交所 (mis.twse.com.tw) 實現低延遲盤中數據。
- **📊 熱度演算法**: 實作基於成交值的動態百分比熱度計算與前端進度條顯示。
